Additional Notes

Navigation

3 bottom tabs: Dashboard, History, Achievements. Use Expo Router with a (tabs) layout.

"Start Session" on Dashboard navigates to a Focus Timer screen (not in this design spec - you build it from scratch). Tapping a session card in History navigates to a Session Detail screen (also your design).

Shadows

Card shadow0 1px 3px rgba(0, 0, 0, 0.05)
CTA button shadow0 4px 14px rgba(108, 92, 231, 0.35)
Phone bezel shadow(not part of app - just the mockup frame)

API data quirks

The API returns session types as deep_focus, quick_sprint, pomodoro (snake_case). The design shows them as title case ("Deep Focus"). You must transform them.

Durations come as milliseconds (durationMs). The design shows "25 min".

See API-SPEC.md in the starter repo for the full endpoint reference.
